Hi, I’m **Ayesha Ansari** 👋

In this task, I worked on building a machine learning model to classify iris flowers into different species based on their features.

---

## 📌 Project Overview

The **Iris dataset** contains measurements of iris flowers such as:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

Using these features, the goal is to **predict the species** of the flower:

* Setosa
* Versicolor
* Virginica

---
